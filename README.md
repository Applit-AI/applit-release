# Applit Desktop

Application de gestion des réunions avec transcription et enregistrement audio.

## Installation

1. Téléchargez le fichier **Applit-Desktop-Setup-x.x.x.exe** depuis la section [Releases](../../releases/latest)
2. Lancez le fichier téléchargé et suivez les étapes de l'installateur
3. L'application se lance automatiquement après l'installation

## Mises à jour

L'application vérifie et installe les mises à jour automatiquement au démarrage.
